# TEAL: Diachronic Poetic Prosody Pipeline

TEAL is a research codebase for analyzing diachronic changes in English poetic prosody and affect (1850-1950).

## Quick Start

1. Create environment and install dependencies:
   - `python -m venv .venv`
   - `source .venv/bin/activate`
   - `pip install -r environment/requirements.txt`
2. Run a pipeline stage:
   - `make ingest`
   - `make clean`
   - `make prosody`
   - `make affect-train`
   - `make affect-infer`
   - `make stats`
   - `make viz`
